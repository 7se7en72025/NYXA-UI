Hey r/reactjs,

I've been working on Falak — a component library focused on **motion and feel**, not just looks. Every component is built with physics-based animations and the sense that UI elements have mass and momentum.

**What's in it:**

- axe-core accessibility testing on every PR
- shadcn/ui registry — `npx shadcn@latest add hero`
- TypeScript strict mode, 45 unit tests, Playwright E2E across 3 browsers
- MIT license
